# Bank Management System (Python)

A simple command-line Bank Management System developed using Python.

## Features
- Create bank account
- Deposit money
- Withdraw money
- Check account balance

## Technologies Used
- Python
- Object-Oriented Programming

## How to Run

1. Clone the repository

git clone https://github.com/yourusername/bank-management-system-python.git

2. Navigate to the folder

cd bank-management-system-python

3. Run the program

python bank_system.py
